How It Works

All colors are derived from HSL base values:

  • --theme-terminal-h: Terminal hue (120 = green)
  • --theme-teal-h: Sheet hue (181 = teal)
  • --theme-yellow-h: Modal hue (48 = yellow)

Click the theme buttons above to see instant color changes!

Notice how ALL UI elements (50+ colors) update simultaneously because they're all derived from the same HSL foundation.
